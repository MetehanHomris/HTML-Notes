ID attributes are essentially used to assign elements a unique ID. ID' are 95% of the time used to specifiy a singular element.
There is no reason to use classes in HTML if there will be no usage of styling, CSS or JavaScript.

While using classes in CSS you need to put a "#" before them to specifiy it is an ID.
